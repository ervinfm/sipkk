CARA INSTALL : 

1. Patikan Sudah Menginstall XAMPP versi php > 7.0.0
2. Start Apache dan MySQL
3. Buat database pada phpMyAdmin dengan nama db_sipk
4. Import database db_sipk ke phpMyAdmin
5. Pindahkan project SIPK ke htdocs 
6. Buka Browser dan masukkan url localhost/sipk
7. Login Admin dengan username : admin | password : admin
8. Login Guru dengan NIP/NIY Guru
9. Enjoy To explore
